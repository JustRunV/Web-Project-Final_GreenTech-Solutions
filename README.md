GreenTech Solutions Website

This project is a multi-page website created for GreenTech Solutoins to showcase eco-friendly technology and promote sustainability.

Features:
- Homepage with company introduction and featured products
- About page with company mission and timeline
- Products page with search/filter functionality
- Blog/Resources page with external sustainability articles
- Contact page with a working form and contact information
- Newsletter signup form
- Responsive design for mobile devices

Technologies Used
- HTML5
- CSS3
- JavaScript

How to Run
1. Download or clone the project files
2. Open the project folder
3. Open `index.html` in a web browser

Notes:
- JavaScript is used for the contact form, product search, and newlsetter interaction.
- The website is designed to be accessible and mobile-friendly.
